Сруктура внутренних папок

Вся информация по оборудованию Экополимер находится по ссылке: H:\Отделы\ОТО\object (OTONEW)\ОБОРУДОВАНИЕ ЭКОПОЛИМЕР Рисунок 1 - структруа папок внутренного каталога
0. ОБЪЕКТЫ
чтение
В данной папке содержиться архив объектов и материалы по ним: исходные данные, чертежи, предложения, расчеты цен для отделов маркетинга. Этов внутренняя папка и доступ к ней имеют только сотрудники ОТО.
1. РЕШЕТКИ, 2. ЗАТВОРЫ, 3. ТРАНСПОРТЕР И УПЛОТНИТЕЛЬ, 4. ПРОЧЕЕ ОБОРУДОВАНИЕ, 5. ОТСТОЙНИКИ, 13. ШУ, 14. АЭРАТОР
запись
Данные папки содержат информацию необходимую для формирования расчетов по оборудованию. (Эти папки только для чтения).
7. ТКП
запись
В данной папке содержатся шаблоны ТКП архивного и нестандартного оборудования (папка полько для чтения).
8. НОВЫЕ ЦЕНЫ
запись
В папке содержится программа расчета и все генерируемые ею файлы: расчеты себестоимости, технико-коммерческие предложения, спецификации, шаблоны, блоки рассчетов.
9. КАТАЛОГИ
запись
В папке содержятся каталоги оборудования Экополимер, а так же материалы подготавливаемые отделом ОТО для технических каталогов производственного предеприятия ПП "Экополимер".
10. ГЧ
запись
В папке содержятся архив установочных чертежей оборудования Экополимер, которые были ранее выданы проектному отделу (вклчая сторонние проектные орнанизации).
11. Опросные листы и ТЗ
запись
В папке содержятся ОЛ на оборудование выданные в проекты, шаблоны технических заданий (ТЗ) для закупки оборудования.
12. Типовые проекты
запись
В папке содержятся архив типовых проектов.
15. АДМИНИСТРАТИВНЫЕ РАБОТЫ
запись
В папке содержятся работы выполняемые для департамента производства: подведение статистики, расчеты коэффициентов....
16. НОРМАТИВНЫЕ ДОКУМЕНТЫ
запись
База нормативно-прававыз документов: СНиП, ГОСТЫ, Законы, постановления, распоряжения, справочники, пособия...
17. ИЭ
запись
